Tutorial - Sistema de Asistencia

Paso 1: Selecciona el Área

Elige el invernadero o área donde estás tomando asistencia. Cada área guarda su asistencia por separado.

Ejemplo: Invernadero 9, Fitosanidad, Riego, etc.

Paso 2: Elige la Fecha

Selecciona la fecha para la cual estás registrando la asistencia. Por defecto se usa la fecha actual.

Puedes registrar asistencia para días anteriores si es necesario.

Paso 3: Registra la Asistencia

Se mostrarán automáticamente los empleados de esa área. Usa los botones "Presente" o "Ausente" para cada uno.

Puedes buscar empleados por nombre o código si necesitas agregar personal de otras áreas.

Paso 4: Acciones Adicionales

Usa los botones de acción para:

  • Mover empleados entre áreas
  • Agregar empleados de otras áreas temporalmente
  • Marcar a todos como presentes rápidamente

Paso 5: Guarda los Cambios

No olvides hacer clic en "Guardar" para registrar toda la asistencia en el sistema.

Si no guardas, se perderán todos los cambios realizados.
